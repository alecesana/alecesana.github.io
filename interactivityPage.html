About

I started coding in C++ while graduating in Materials Science, a subject involving Physics, Chemistry and Mathematics applied to the development of new materials. I have been working in sales of electronics systems for the past four years and skateboarding since 2001. In my spare time I have been designing, assembling and programming various types of electronics systems as a private contractor. Along the way I picked up more languages and frameworks to suit different needs, such as

  • more C++ (Openframeworks, OpenCV, some code here)
  • OpenGL and WebGL
  • Java (Processing, LXstudio)
  • Python
  • Mql4
  • Scio Molecular Spectroscopy Dev Kit
  • Brand specific programmable logic controller languages (Arduino, Splat controls, Allen Bradley)
  • Html, CSS, and JavaScript (Vanilla and JQuery, currently studying React and other Front-End frameworks)
