About

I started coding in C++ while graduating in Materials Science, an interdisciplinary field of study involving Chemistry, Physics and Mathematics applied to the development of new materials (link to Wikipedia Materials Science Page , if you'd like to know more about it). I have been working in sales of electronics systems for the past three years. In my spare time I have been designing, assembling and programming electronics systems as a private contractor. Along the way I picked up more languages and frameworks to suit various projects needs, such as

  • Java (Processing)
  • C++ (Openframeworks, OpenCV)
  • Python (Raspberry Pi)
  • Mql4
  • Scio Molecular Spectroscopy Dev Kit
  • Brand specific programmable logic controller languages (Arduino, Splat controls, Allen Bradley)
  • Html, CSS, and JavaScript (Vanilla and JQuery, currently studying React and other Front-End frameworks)
